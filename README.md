# CookFlow CRM (v2.5)

## Enterprise-Grade Culinary Orchestrator
CookFlow CRM is a multi-agent cooking management system designed to eliminate decision fatigue and budget friction. By bridging the gap between active calendar schedules, dietary wellness, and strict financial disciplines, CookFlow transforms daily meal preparation into an optimized, data-driven workflow.

---

## 🎯 Core Objectives
* **Dynamic Schedule Correlation:** Aligns meal prep windows with your actual calendar (gym, meetings, commutes).
* **INR Localized Budgeting:** An active "Cost Reduction Loop" that scans ingredients against user-defined daily budgets.
* **Multi-Agent Intelligence:** Uses a synchronized four-agent pipeline to generate recipes, calculate macros, optimize grocery lists, and manage allergen substitutions.

---

## 🛠️ The Tech Stack

### Frontend UI Engine
* **Framework:** React 19 + Vite
* **Styling:** Tailwind CSS (Theme: Deep Sage `#4E6E58`, Warm Ochre `#E89942`, Crisp Off-whites `#FAFAFA`)
* **Interaction:** Framer Motion for fluid transitions and terminal simulations.
* **Icons:** Lucide React

### Backend & AI Orchestration
* **Server:** Express.js (Node.js/TypeScript)
* **AI Core:** Google Gemini (`gemini-3.5-flash`)
* **State Machine:** Sequential dependency chain via LangGraph logic.
* **Telemetry:** Built-in TikToken cost tracking for real-time API expense monitoring.

---

## 🧭 Navigation Workspace
* **Kitchen Dashboard:** Central command node for daily budgets and macro-balanced menu overviews.
* **Day Scheduler:** Regional presets (e.g., *Mumbai Local Commuter*, *Delhi WFH Family*) with granular budget/allergy controls.
* **Culinary Board:** Interactive step-by-step instructions with live "Stove Alert" countdown timers.
* **Smart Grocery List:** Aggregates ingredients across all meals to minimize waste and redundant shopping.
* **Substitution Ledger:** Real-time parsing for allergen intercepts and cost-effective alternatives.
* **Financial Audit:** Comparative analysis between retail pricing and local *Mandi* (market) variables.
* **Agent Trace Console:** Live telemetry and terminal-style logs showing the Multi-Agent Orchestrator in action.

---

## 🚀 Quick Start Guide
1.  **Configure:** Navigate to the **Day Scheduler** and select a lifestyle preset.
2.  **Orchestrate:** Hit the **"Trigger Multi-Agent Orchestrator"** button.
3.  **Monitor:** Watch the **Agent Trace Console** for real-time pipeline status.
4.  **Execute:** Utilize the **Kitchen Dashboard** and **Stove Alerts** to manage your cooking workflow seamlessly.
